# Contributing

Thank you for your interest in improving this automation framework! To contribute:

1. **Fork** the repository and create a feature branch.
2. Write clear, well-documented code and add tests where applicable.
3. Follow the existing project structure and naming conventions.
4. Commit with descriptive messages and open a pull request.
5. Ensure the GitHub Actions workflow passes (`npx playwright test`).

> Pull requests should be targeted at the `main` branch.

Feel free to open issues for bugs or feature requests.
